# 컴포넌트 라이브러리 (Components)

UI 화면에서 반복적으로 등장하는 요소(버튼 모형, 레이아웃 컨테이너, 사이드바, 알림 팝업 등)를 **독립적인 단위로 컴포넌트화 시켜 둔 조각보 모듈**입니다. 이 폴더의 코드들은 비즈니스 알고리즘보단 "재사용 가능한 시각적 표현"에 중점을 둡니다.

## 주요 포함 내역
- `Layout.jsx`: 앱 전역에 걸친 레이아웃 구조체입니다. 최상단 Auth Guard(인증 여부 1차 체크 후 `<Navigate to="/login" />`) 로직을 감싸고 있어 모든 서브페이지 화면 진입점에 깜빡임 없는 보안 방어벽 역할을 수행합니다.
- `Sidebar.jsx`: 좌측 네비게이션 트리. 활성화된 상태의 라우터를 표시하며, 로그인 서버의 속성 조회를 통해 하단에 로그아웃 버튼을 그립니다.
- `Toast.jsx`: 우측 상단이나 하단에 잠시 등장했다가 애니메이션과 함께 사라지는 (Fade-Out) 상태 알림 메세지(성공/에러) Context Provider를 관리하는 글로벌 Hooks 생태계입니다.
